Como eu posso ajudar?

Há muitas maneiras de ajudar se estiver interessado em ver a web crescer e ser ainda mais útil...

Coloque alguns dados
Há muitas maneiras de fazer isso. A web precisa de dados brutos - hipertexto fresco ou arquivos de texto simples antigos, ou servidores inteligentes que fornecem visualizações de bancos de dados existentes. Veja mais detalhes, etiqueta.
Sugira a alguém fazer
Talvez você conheça um sistema que seria legal ter na web. Que tal sugerir ao envolvido que monte um servidor W3?
Gerenciar um assunto
Se você sabe algo sobre o que está acontecendo em um determinado campo, organização ou país, gostaria de manter uma visão geral atualizada dos dados on-line?
Escreva algum software
Temos uma grande lista de coisas a serem feitas. Ajude você mesmo - todas as contribuições serão recebidas com gratidão! veja a lista.
Envie-nos sugestões
Adoramos receber e-mails... www-bug@info.cern.ch
Diga a seus amigos
Instale/instale-se o software cliente em seu site. Cite as coisas pelo endereço W3 para permitir que os usuários do w3 as peguem diretamente.
